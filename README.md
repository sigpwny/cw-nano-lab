# ChipWhisperer Nano Lab

This repository contains basic challenges for the ChipWhisperer Nano platform. It is NOT intended as a replacement for the official ChipWhisperer tutorials, but rather as a simplified, quick-start.

Check out the official ChipWhisperer tutorial at [newaetech/chipwhisperer-jupyter](https://github.com/newaetech/chipwhisperer-jupyter)!

## Setup

You will first need to install the Python dependencies.

```bash
pip install -r requirements.txt
```
